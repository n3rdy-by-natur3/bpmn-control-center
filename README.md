# BPMN Control Center
Monitoring the processes of your process engine (starting with Camunda, Flowable is planned). 

## Vue.js
### Project Setup

```sh
npm install
```

#### Compile and Hot-Reload for Development

```sh
npm run dev
```

#### Compile and Minify for Production

```sh
npm run build
```
